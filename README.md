# ZimConnect

ZimConnect is being rebuilt as a marketplace platform with:

- `Next.js` on the frontend
- `Django` on the backend

The current repository contains the early frontend shell and brand direction for the product.

## Frontend Rebuild Documentation

The main handoff document for the frontend developer is here:

- [docs/frontend-rebuild-spec.md](docs/frontend-rebuild-spec.md)

It covers:

- product scope
- required pages and routes
- user flows
- reusable components
- suggested API contracts for Django
- implementation phases

## Local Development

Run the frontend locally with:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in the browser.
